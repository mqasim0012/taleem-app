HOW TO RUN:

Install: php, mysql, phpMyAdmin, Apache service unless available.

Perform the following on a terminal:

  git clone git@github.com:mqasim0012/taleem-app.git

  cd taleem-app
  
  php -S localhost:8000       (OR any other port)
